# 📊 E-Commerce Customer Behavior & Revenue Intelligence Dashboard

An end-to-end business analytics project focused on analyzing customer behavior, revenue trends, payment patterns, and operational performance using Python and Power BI.

This project was built using a real-world e-commerce dataset and follows a complete analytics workflow — from raw data cleaning to interactive dashboard development and business insight generation.

---

# 🚀 Project Objectives

- Analyze customer purchasing behavior
- Identify revenue trends and operational patterns
- Track delivery performance and customer satisfaction
- Build an interactive business intelligence dashboard
- Generate actionable business insights from raw datasets

---

# 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Power BI
- Data Cleaning & Transformation
- Business Intelligence & Data Visualization

---

# 📂 Dataset Information

The project combines multiple e-commerce datasets, including:

- Orders Dataset
- Customers Dataset
- Payments Dataset
- Reviews Dataset
- Order Items Dataset

The datasets were cleaned, transformed, and merged into a single analytical dataset for dashboard development.

---

# ⚡ Data Preparation Workflow

## Key Steps Performed

- Removed duplicate records
- Handled missing values
- Converted timestamps into datetime format
- Merged multiple datasets using common keys
- Created KPIs and analytical measures
- Generated monthly revenue and operational insights

---

# 📈 Dashboard Features

## Executive Overview
- Total Revenue
- Total Orders
- Average Review Score
- Delivery Performance
- Monthly Revenue Trends
- Revenue Distribution by State

## Customer & Operational Insights
- Revenue Impact by Order Status
- Customer Satisfaction Analysis
- Payment Method Insights
- Order Fulfillment Distribution
- Monthly Operational Heatmap

---

# 🔍 Key Business Insights

- Generated over **$16.6M revenue** across 99K+ orders
- Achieved a **97% successful delivery rate**
- São Paulo contributed the highest revenue share
- Customer review scores averaged above **4.0**
- Revenue peaked significantly during late 2017
